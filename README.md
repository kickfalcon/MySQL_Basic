# MySQL_Basic

Códigos desarrollados durante el curso de Introduccion a SQL con MySQL, donde se abordaron temas como la historia de SQL y de MySQL, además, de la instalación del IDE de MySQL.

## Conceptos aprendidos
- Tipos de Variables
- Función de Primary Key
- Generar Tablas
- Agregar, Modificar y Borrar Registros
- Modificar Tablas
- Filtros básicos y compuestos
